# destiny.gg-emotes

JSON file of destiny.gg emotes.

## Example

``` javascript
var emotes = require('destiny.gg-emotes');
// => ['DRAVEWIN', ..., 'nathanFeels']
```

## Installation

``` bash
$ npm install destiny.gg-emotes
```

## API

``` javascript
var emotes = require('destiny.gg-emotes');
```

### `emotes`

_Array_ of destiny.gg emotes.
